# Interest Rate Card

Annual interest rate (reducing balance) by CIBIL band. Salaried applicants get a 25 bps concession over self-employed.

| CIBIL Band | Salaried (p.a.) | Self-employed (p.a.) |
|---|---|---|
| 780+ (Excellent) | 8.50% | 8.75% |
| 750–779 (Very Good) | 8.75% | 9.00% |
| 720–749 (Good) | 9.25% | 9.50% |
| 700–719 (Fair) | 9.75% | 10.00% |
| 670–699 (Below Average) | 10.50% | 10.75% |
| 650–669 (Sub-prime) | 11.50% | 11.75% |
| Below 650 | Manual underwriting only |

Rate is fixed for the loan tenure. Floating rate option available for tenures ≥ 60 months at +25 bps initial concession.
