# Processing Fees & Charges

## Processing fee
- Standard: **0.5% of loan amount + GST**, minimum ₹2,500, maximum ₹10,000
- Premium customers (CIBIL ≥ 780, salaried at A-list employer): **0.25% + GST**
- Festival offers periodically waive processing fees on new car loans

## Other charges
| Charge | Amount |
|---|---|
| Stamp duty | As per state, typically 0.1–0.3% of loan |
| Documentation charges | ₹1,000 + GST |
| RTO hypothecation charges | At-actuals (state RTO rate) |
| Insurance facilitation | Optional — no charge from us, customer pays insurer directly |
| Cheque bounce charge | ₹500 + GST per instance |
| Late payment charge | 2% per month on overdue EMI |

All charges disclosed in the Key Facts Statement (KFS) before loan disbursal — RBI mandatory.
