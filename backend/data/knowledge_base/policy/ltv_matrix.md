# Loan-to-Value (LTV) Caps

The maximum loan we extend is a percentage of the on-road price (new) or valuation (used).

| Vehicle | LTV cap |
|---|---|
| New car, ex-showroom price ≤ ₹15L | Up to 90% |
| New car, ex-showroom price > ₹15L | Up to 85% |
| Used car, age ≤ 3 years | Up to 80% |
| Used car, age 3–5 years | Up to 70% |
| Used car, age > 5 years | Case-by-case, typically ≤ 60% |

For premium segment (> ₹40L) applicants must demonstrate liquid net worth ≥ 30% of loan amount. LTV is capped further if CIBIL < 700 — reduce by 5 percentage points per band.
